# nbcamp_personalProject_04  
개인과제로 횡스크롤 슈팅 게임을 만들어 보았습니다.  

## 구현한 기능  
- 플레이어 이동, 더블 점프
- 플레이어 및 적 공격, 피격 구현
- 실시간 점수 표시  
- 게임 경과 시간에 따른 적 소환 방식 변경
- 일시 정지 및 설정 팝업  
- 게임 오버 팝업 및 현재 점수와 최고 점수 표시
